# Blynk field logger

This repo holds the **sensor dashboard** (web app for the field logger) plus project docs, Arduino sketches, and diagrams.

## Layout

| Folder / file | Purpose |
|---------------|---------|
| **sensor-dashboard/** | Web app: HTTP ingest API and dashboard UI. See `sensor-dashboard/README.md` and `sensor-dashboard/WEBSITE_GUIDE.md`. |
| **docs/** | Project notes, papers, guides (markdown and text). |
| **arduino/** | Arduino/ESP8266 sketches (e.g. Blynk virtual pin code). |
| **diagrams/** | Circuit/schematic diagrams (SVG, PNG, and `circuit_diagram.py`). |
| **scripts/** | One-off scripts (e.g. export SVG to PNG, Composio helpers). |

Run the dashboard from `sensor-dashboard/` (see its README).
